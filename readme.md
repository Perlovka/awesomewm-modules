## Installation

Clone repo somewhere:
```shell
cd /tmp
git clone https://github.com/Perlovka/awesomewm-modules.git

```
Copy files to the directory where awesome rc.lua is located:
```shell
cp -ar /tmp/awesomewm-modules/gears ~/.config/awesome/
```
